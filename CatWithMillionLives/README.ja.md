# 概要

新しい信仰"魂のミリオンライヴズ"を追加します。
この信仰はあなたやペットを死から遠ざけます。

[![en](https://img.shields.io/badge/lang-en-red.svg)](https://github.com/Seacolor/ElinMOD/blob/main/CatWithMillionLives/README.md)

# 詳細

## 好物

- 死体
- かつおぶし
- 奉納酒
- チューリュ

## 信仰ボーナス

- 速度
- 魔力
- 隠密
- 窃盗
- 解剖学

## 専用アビリティ

- 反魂

### 反魂

- 仲間を死者の世界から呼び戻す

## 信仰のご褒美

1. 化け猫のナインライヴズ（ペット）
2. 猫の鈴（アーティファクト）

### 化け猫のナインライヴズ

- 仙狸、ふさふさの尻尾、ブースト、浮遊のフィートを持っています
- 仙狸のフィートにより成長すると尻尾が増えて速度とマナと火炎耐性が上昇します（最大7本まで）
- 炎の光線、炎の球、炎の剣、復活を使います

### 猫の鈴

- あなたを死の懲罰から守るファクションエンチャントを持っています

## 祭壇

この MOD を導入すると、野原やネフィアに置かれている祭壇の中に魂の祭壇が増えます。
