# 概要

新しい宗教"罠のカカ・ルー"を追加します。
この宗教は罠を最大限に活用する術を授けます。

[![en](https://img.shields.io/badge/lang-en-red.svg)](https://github.com/Seacolor/ElinMOD/blob/main/CatChingOfTraps/README.md)

# 詳細

## 好物

- 死体
- 石
- 奉納酒

## 信仰ボーナス

- 感覚
- 探索
- 罠解体

## 特殊能力

- 罠を見ることを可能にする
- ネフィアの罠の数にボーナス

## 信仰のご褒美

[spoiler]

1. カラクロイド（ペット）
2. 罠師の腕輪（アーティファクト）

### 化け猫のナインライヴズ

- 魔導生命体1のフィートを持っています
- グラビティ、罠設置を使います
- 罠設置はランダムな罠を足元に設置します

### 罠師の腕輪

- 罠の視認と回避と利用を可能にするファクションエンチャントを持っています

[/spoiler]

## 祭壇

この MOD を導入すると、野原やネフィアに置かれている祭壇の中に罠の祭壇が増えます。
