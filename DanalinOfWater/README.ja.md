# 概要

新しい信仰 **"水のダナリン"** を追加します。
この信仰は週によって信仰ボーナスが変動し、狂気度の上昇によってボーナスが増加します。

[![en](https://img.shields.io/badge/lang-en-red.svg)](https://github.com/Seacolor/ElinMOD/blob/main/DanalinOfWater/README.md)

# 詳細

## 好物

- 死体
- 体のパーツ

## 信仰ボーナス

- 狂気度

他の信仰ボーナスは週によって変動します。

また、狂気度が高いほど信仰ボーナスが大きくなり、さらに追加効果が発動します。

### 狂気度による追加効果

- 10: 信仰感知
- 20: 狂気による朦朧・混乱・恐怖の無効
- 30: 透明可視
- 40: 水中呼吸
- 50: 星界からの恐怖 2

## 信仰のご褒美

1. 深きもの（ペット）
2. 海のお守り（アーティファクト）

### 深きもの

- 星界からの恐怖と透明可視と水中呼吸のフィートを持っています
- アンデッドとしても扱われます
- 狂気の眼差しを使います

### 海のお守り

- 水中呼吸、魅力、水泳のエンチャントを持っています
- 各製作技能のファクションを持っています

## 祭壇

この MOD を導入すると、野原やネフィアに置かれている祭壇の中に水の祭壇が増えます。
